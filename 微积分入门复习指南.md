## 1. 什么是导数？——「变化率」的直觉

**导数回答一个问题：当输入变化一点点时，输出变化多少？**

举个例子：你开车从北京去上海。

- 路程 s(t) 随时间 t 变化
- **速度**就是路程对时间的导数：v = ds/dt

如果你在第 2 小时的时候速度是 100 km/h，意思就是：**在那一瞬间**，时间每增加 1 小时，路程大约增加 100 km。

数学表达：

```
f'(x) = lim    [ f(x + Δx) − f(x) ] / Δx
        Δx→0
```

别被这个式子吓到。翻译成人话：

> f'(x) = 当 x 增加一个**极小极小的量**时，f(x) 的变化量 ÷ 那个极小量

在深度学习中，我们关心的永远是：

> **如果我把这个参数调大一点点，损失函数会变大还是变小？变多少？**

## 2. 导数的几何意义：切线斜率

- 函数曲线在某一点的**切线斜率** = 该点的导数
- 斜率为正 → 函数在上升
- 斜率为负 → 函数在下降
- 斜率为 0 → 函数在局部最高点或最低点

```
        y
        ^
        |      /
        |     /  ← 切线，斜率 = f'(x₀)
        |    ●
        |   /  ← 曲线 y = f(x)
        |  /
        | /
        |/
        +------------------> x
              x₀
```

深度学习的目标就是：**找到让损失函数最小的参数，也就是斜率为 0 的那个点。**

## 3. 基本求导公式（背下来）

以下是你必须烂熟于心的公式。好消息是，深度学习中真正高频出现的就那么几个。

| 函数 f(x) | 导数 f'(x) | 记忆口诀 |
|:---:|:---:|:---|
| c（常数） | 0 | 常数不变，导数为 0 |
| x | 1 | x 对自己变化率就是 1 |
| xⁿ | n·xⁿ⁻¹ | 指数搬下来，指数减 1 |
| eˣ | eˣ | **神奇！eˣ 的导数就是自己** |
| ln x | 1/x | 对数导数是倒数 |
| aˣ | aˣ·ln a | 一般指数函数，多乘个 ln a |
| sin x | cos x | 正弦导数是余弦 |
| cos x | −sin x | 余弦导数是负正弦（注意负号！） |

**深度学习中最常用的三个**：xⁿ 的导数、eˣ 的导数、ln x 的导数。

## 4. 求导法则：加法、乘法、链式法则

### 4.1 加法法则（最简单）

```
( f(x) + g(x) )′ = f′(x) + g′(x)
```

> 和的导数 = 导数的和。直接拆开分别求就行。

**常数倍法则**（加法的特例）：

```
( c·f(x) )′ = c·f′(x)
```

> 常数可以提到外面。

### 4.2 乘法法则

```
( f(x)·g(x) )′ = f′(x)·g(x) + f(x)·g′(x)
```

> 前导后不导 + 前不导后导

### 4.3 链式法则（最重要！深度学习的基石）

```
d/dx [ f(g(x)) ] = f′(g(x)) · g′(x)
```

> **外层函数求导（保持内层不变） × 内层函数求导**

**示例**：求 h(x) = (3x + 1)² 的导数

- 外层 f(u) = u²，导数 f′(u) = 2u
- 内层 g(x) = 3x + 1，导数 g′(x) = 3
- 链式法则：h′(x) = 2(3x + 1) · 3 = 6(3x + 1) = 18x + 6

**示例**：求 h(x) = e²ˣ 的导数

- 外层 f(u) = eᵘ，导数 = eᵘ（保持不变）
- 内层 g(x) = 2x，导数 = 2
- 结果：h′(x) = e²ˣ · 2 = 2e²ˣ

> **为什么链式法则在深度学习中如此重要？**
>
> 神经网络是函数的嵌套：
>
> ```
> output = fₙ( fₙ₋₁( ... f₁(x) ) )
> ```
>
> 链式法则让你可以一层一层地"往回传"导数——这就是**反向传播**。

## 5. 偏导数：多元函数的导数

深度学习中的函数从来都不是一个输入。损失函数依赖于**成千上万个参数**。

**偏导数回答：当我只改变其中一个变量（其他变量固定不变）时，函数变化多快？**

符号：对 x 的偏导数写为 ∂f/∂x（用弯的 ∂ 代替直的 d）

**示例**：f(x, y) = x² + xy + y³

- 对 x 求偏导（把 y 当常数）：∂f/∂x = 2x + y
- 对 y 求偏导（把 x 当常数）：∂f/∂y = x + 3y²

**规则极其简单**：对谁求偏导，就把其他所有变量当成常数，然后用一元函数的求导规则。

再多看一个例子：

f(x, y) = x · eʸ

- ∂f/∂x = eʸ（eʸ 被当作"常数"）
- ∂f/∂y = x · eʸ（x 是"常数"，eʸ 的导数还是 eʸ）

## 6. 梯度：所有偏导数组成的向量

**梯度 = 把所有偏导数打包成一个向量**

```
∇f = ( ∂f/∂x₁ , ∂f/∂x₂ , ... , ∂f/∂xₙ )
```

梯度有两个关键性质：

### 性质 1：梯度指向函数**上升最快**的方向

站在山坡上，梯度指向最陡的上坡方向。

### 性质 2：负梯度指向函数**下降最快**的方向

**← 这就是梯度下降的核心思想！**

```
        梯度指向 ↑ （最陡上坡）
             /
            /
           ●  ← 你在这里
            \
             \
              负梯度指向 ↓ （最陡下坡）
```

## 7. 梯度下降：深度学习的核心优化方法

### 问题设定

- 你有一个损失函数 L(**w**)，它依赖于参数 **w** = (w₁, w₂, ..., wₙ)
- 你想找到让 L 最小的参数值
- 但参数可能有成千上万个，没法画图找最低点

### 梯度下降的直觉

想象你在浓雾中站在一座山上，想下到山谷：

1. 你感受脚下的坡度（= 计算梯度）
2. 朝最陡的下坡方向走一小步（= 沿负梯度方向更新参数）
3. 在新位置重新感受坡度
4. 重复，直到坡度接近零（= 到达谷底）

### 数学公式

```
w_new = w_old − η · ∇L(w_old)
```

每个符号的含义：

| 符号 | 含义 | 说明 |
|:---|:---|:---|
| **w** | 所有参数组成的向量 | 比如神经网络中所有的权重 |
| ∇L | 损失函数的梯度 | 每个参数对损失的影响有多大 |
| η | 学习率（eta） | 控制每一步走多大 |
| −（减号） | 沿负梯度方向 | 梯度指向上坡，我们要下坡，所以取反 |

### 为什么必须理解这个？

**训练神经网络 = 反复执行梯度下降**

你不需要手算梯度（PyTorch/TensorFlow 会自动求导），但你**必须理解**：
- 梯度告诉我们参数该往哪个方向调
- 学习率太大 → 跳过头，无法收敛
- 学习率太小 → 收敛太慢

```python
# PyTorch 中的梯度下降（让你感受一下）
for epoch in range(num_epochs):
    loss = compute_loss(model, data)  # 前向传播，算损失
    loss.backward()                   # 反向传播，自动计算梯度
    for param in model.parameters():
        param.data -= lr * param.grad  # 梯度下降更新参数
        param.grad.zero_()             # 清零梯度，准备下一轮
```

## 8. 反向传播的本质：链式法则的工程实现

### 为什么叫"反向"？

神经网络的计算是**前向**的：

```
x → f₁ → a₁ → f₂ → a₂ → ... → fₙ → output → Loss
```

但梯度计算是**反向**的，从 Loss 往回传：

```
∂L/∂x = ∂L/∂aₙ · ∂aₙ/∂aₙ₋₁ · ... · ∂a₁/∂x
```

这就是链式法则！每一层只需要知道两样东西：
1. 自己这一层"输出对输入的导数"
2. 上一层传来的梯度

两者相乘，再传给下一层。

### 一个具体例子

假设一个最简单的两层"网络"：

```
输入 x → 乘以 w → y = wx → 损失 L = (y − target)²
```

我们要算 ∂L/∂w（损失对参数 w 的敏感度）：

**用链式法则拆解**：

```
∂L/∂w = ∂L/∂y · ∂y/∂w
```

一步步来：
- L = (y − t)²，所以 ∂L/∂y = 2(y − t)
- y = w·x，所以 ∂y/∂w = x
- 合起来：∂L/∂w = 2(y − t) · x

> 你看，从 Loss 开始，一层一层往回乘，这就是反向传播的全部奥秘。

## 9. 常见函数的梯度速查表

深度学习中你实际会遇到的梯度：

| 操作 | 前向公式 | 反向梯度 | 出现场景 |
|:---|:---|:---|:---|
| 线性层 | y = wx + b | ∂y/∂w = x, ∂y/∂x = w | 全连接层 |
| Sigmoid | σ(x) = 1/(1+e⁻ˣ) | σ(x)·(1−σ(x)) | 二分类输出 |
| Tanh | tanh(x) | 1 − tanh²(x) | RNN |
| ReLU | max(0, x) | 0（当 x<0）, 1（当 x>0） | 激活函数 |
| Softmax+交叉熵 | 复合操作 | ŷ − y（极其简洁！） | 多分类输出 |
| 均方误差 | (y − t)² | 2(y − t) | 回归任务 |
| 向量点积 | **w**ᵀ**x** | 对 w 的梯度是 **x** | 线性组合 |

> **最重要的一个**：Softmax + 交叉熵损失的梯度恰好是 `预测值 − 真实标签`。这个简洁的结果是深度学习中最优美的公式之一。

## 10. 总结：你真正需要记住的

### 核心概念（按重要性排序）

1. **导数 = 变化率** —— 输入变一点点，输出变多少
2. **偏导数** —— 多元函数中，只对一个变量求导，其他当常数
3. **梯度** —— 所有偏导数组成的向量，指向最陡上坡方向
4. **链式法则** —— 复合函数求导：外层导 × 内层导
5. **梯度下降** —— 沿负梯度方向走小步，逐步逼近最小值
6. **反向传播** —— 用链式法则从 Loss 往回计算每一层参数的梯度

### 必须背下的公式

```
(xⁿ)′ = n·xⁿ⁻¹      ← 幂函数求导
(eˣ)′ = eˣ           ← 指数函数求导（导数就是自己！）
(ln x)′ = 1/x        ← 对数函数求导
```
```
链式法则： d/dx[ f(g(x)) ] = f′(g(x)) · g′(x)
```

### 符号速查

| 符号 | 读法 | 含义 |
|:---|:---|:---|
| f′(x) 或 df/dx | "f 撇 x" 或 "df dx" | 一元函数的导数 |
| ∂f/∂x | "partial f partial x" | 多元函数的偏导数 |
| ∇f | "nabla f" 或 "梯度 f" | 梯度（所有偏导数组成的向量） |
| η | "eta" | 学习率 |

### 不要做的事

- ❌ 不要在学深度学习前去啃一整本微积分教材
- ❌ 不要试图手算复杂网络的梯度（交给框架）
- ❌ 不要把符号推导和直觉理解混淆——理解「梯度指向最陡方向」比会推导重要 100 倍

### 正确的学习路径

1. 把这份文档看懂（尤其是链式法则和梯度下降这两节）
2. 打开《动手学深度学习》，看到公式时试着理解「这是在求哪个变量的梯度」
3. 用 PyTorch 写代码时，关注 `loss.backward()` 这行——它在帮你做链式法则
4. 遇到不理解的导数，回来看第 9 节的速查表
